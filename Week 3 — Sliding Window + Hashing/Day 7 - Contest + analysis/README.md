# Day 7: Weekly Contest + Analysis

## Overview
It's time to test your synthesis under pressure. This 2-hour virtual contest simulates a standard Codeforces Div 2 / LeetCode array problem setting. The difficulty strictly increases from A (500 rating) to F (3000 rating).

## Rules
1. **No peeking.** Do not look at the `solutions.cpp` or your previous notes.
2. **Timebox.** Force yourself to stop after 120 minutes.
3. **Analyze.** If you missed a problem, read the editorial inside `solutions.cpp` and implement it from scratch.

## Files
| File | Purpose |
|---|---|
| `problems.md` | Match constraints and descriptions (Problems A-F) |
| `solutions.cpp` | Codeforces-style editorials and clean solutions |
| `todo.md` | Contest structure and reflection sheet |
| `index.html` | Progress tracker and time simulator |
