# Day 6: Pattern Revision

## Overview
This is the synthesis test. Interviewers will never say "Use a hash map to fix this array". They will give you constraints and a problem loop, and you must rapidly categorize the structure. This day reinforces cross-pattern application: inversions, mathematical Exact calculation, and debugging.

## Core Concepts
1. **The Four Pillars**: Fixed Window, Variable Window, Flat HashMap, Prefix HashMap.
2. **The `Exact=AtMost-AtMost(-1)` paradigm**.
3. **The Inverse Window**: Operating on array borders equals operating on the central subarray!
4. **Bitmasking Sliding Windows**: Using XOR to reverse `OR` states since overlapping bits are zero.
