# Day 7: Week 2 Capstone Contest

## Contest Overview
This is a 2.5-hour simulated Codeforces-style contest focused entirely on Two Pointers, In-Place Array Manipulation, Dutch National Flag, Matrix Trickery, and Array Frequency/Cycle patterns. 

You must apply the O(1) space constraints and optimal O(n) logic learned throughout Week 2.

## Problem Set
1. **Problem A:** Remove Duplicates II (900)
2. **Problem B:** Sort Colors (1000)
3. **Problem C:** Subarray Product Less Than K (1200)
4. **Problem D:** Rotate Image (1300)
5. **Problem E:** Meeting Rooms II (1400)
6. **Problem F:** Minimum Window Substring (1600)
7. **Problem G:** First Missing Positive (1800)
