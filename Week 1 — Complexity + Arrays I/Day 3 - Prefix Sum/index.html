📖 Block 1 — 1D Foundations

⏱️ 3 Hours

Understand the prefix construction equation.

The prefix array stores the cumulative sum up to index i.

prefix[i] = prefix[i-1] + a[i]

Why? Creating it takes O(N) time initially, but subsequently, finding the sum of ANY subarray from L to R takes O(1).

Extracting a subsegment using arithmetic subtraction.

To find the sum of original array from index L to R, we take all the sum up to R, and subtract the sum of elements right before L.

Sum(L, R) = prefix[R] − prefix[L-1]

Handling Edge Case: L=0

If L is 0, L-1 is -1 which causes an Array Out of Bounds Exception. In C++, we often use a 1-Indexed Prefix Array of size N+1 where prefix[0] = 0.

Class instantiation and multiple query answering.

class NumArray {
    vector<int> pref;
public:
    NumArray(vector<int>& nums) {
        pref.resize(nums.size() + 1, 0); // 1-indexed to avoid L-1 issues
        for(int i=0; i<nums.size(); i++) {
            pref[i+1] = pref[i] + nums[i]; 
        }
    }
    
    int sumRange(int left, int right) {
        // Since pref is 1-indexed, right becomes right+1, left becomes left
        return pref[right + 1] - pref[left];
    }
};

🧠 Block 2 — Hash Map Integration

⏱️ 4 Hours
Golden Rule: Whenever a problem asks for "Subarrays summing to a condition", you almost ALWAYS need Prefix Sums combined with a HashMap.

Count total number of continuous subarrays whose sum equals K. (O(N) Time)

The equation turns into: prefix[curr] - target_prefix = K → target_prefix = prefix[curr] - K.

int subarraySum(vector<int>& nums, int k) {
    int count = 0, sum = 0;
    unordered_map<int, int> freq;
    freq[0] = 1; // Base case: an empty subarray matches prefix total identically

    for(int num : nums) {
        sum += num;
        if(freq.count(sum - k)) {
            count += freq[sum - k]; // We found identical previous prefixes
        }
        freq[sum]++;
    }
    return count;
}

Find the max length of a contiguous subarray with an equal number of 0s and 1s.

Change of Base: If we treat 0 as -1, the problem mathematically transforms into: "Find the longest subarray with a sum exactly equal to 0". 
int findMaxLength(vector<int>& nums) {
    unordered_map<int, int> map; // maps prefixSum to first index seen
    map[0] = -1; // baseline
    int maxLen = 0, sum = 0;
    
    for(int i = 0; i < nums.size(); i++) {
        sum += (nums[i] == 0) ? -1 : 1;
        
        if(map.count(sum)) {
            maxLen = max(maxLen, i - map[sum]);
        } else {
            map[sum] = i; // only save first occurrence to maximize length
        }
    }
    return maxLen;
}

🟧 Block 3 — 2D Scaling

⏱️ 4 Hours

How to build a 2D cumulative matrix correctly.

To build 2D `pref[r][c]`, we take the row above, the column left, subtract the overlapping diagonal, and add current cell a[r][c].

pref[r][c] = A[r][c] + pref[r-1][c] + pref[r][c-1] - pref[r-1][c-1]

To Query 2D sum from (r1, c1) to (r2, c2):

Q = pref[r2][c2] - pref[r1-1][c2] - pref[r2][c1-1] + pref[r1-1][c1-1]

Implementing LeetCode 304 smoothly.

class NumMatrix {
    vector<vector<int>> dp;
public:
    NumMatrix(vector<vector<int>>& m) {
        if(m.empty() || m[0].empty()) return;
        int R = m.size(), C = m[0].size();
        dp = vector<vector<int>>(R + 1, vector<int>(C + 1, 0)); // 1-indexed DP
        
        for(int r = 0; r < R; r++) {
            for(int c = 0; c < C; c++) {
                dp[r+1][c+1] = dp[r][c+1] + dp[r+1][c] 
                             - dp[r][c] + m[r][c];
            }
        }
    }
    
    int sumRegion(int r1, int c1, int r2, int c2) {
        return dp[r2+1][c2+1] - dp[r1][c2+1] 
             - dp[r2+1][c1] + dp[r1][c1];
    }
};

🔥 Block 4 — Advanced Constraints

⏱️ 3 Hours

Math constraint: `(P_curr - P_target) % K = 0` implies `P_curr % K == P_target % K`.

If the remainder of the prefix sum of j divided by K is the same as the remainder of the prefix sum of i divided by K, then the subarray between i and j is completely divisible by K.

int subarraysDivByK(vector<int>& nums, int k) {
    unordered_map<int, int> modFreq;
    modFreq[0] = 1; // Base condition
    int sum = 0, count = 0;
    
    for(int num : nums) {
        sum += num;
        // Proper modulo handling for negative numbers in C++
        int rem = ((sum % k) + k) % k; 
        
        count += modFreq[rem]; // If we've seen this remainder, pair them!
        modFreq[rem]++;
    }
    return count;
}
