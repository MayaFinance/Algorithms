📚 Block 1 — Architecture

⏱️ 3 Hours

The physical difference between Stack and Heap limits.

Stack (Static Memory)

Memory is managed by the CPU automatically. It is extremely fast, contiguous, but strictly limited (typically ~8MB on Linux). Trying to allocate int arr[10000000] will cause a Stack Overflow.

Heap (Dynamic Memory)

Requested via OS system calls. Extremely large limits depending on RAM. Slower allocations.

int* dynamic_arr = new int[n]; // C++ Heap Allocation
// Must be manually freed to avoid memory leaks
delete[] dynamic_arr;

The magic behind O(1) random access in arrays.

Arrays achieve O(1) retrieval because they don't search. The CPU mathematically calculates the memory address instantly.

Target_Address = Base_Pointer + (Index × sizeof(Data_Type))

Example: If an integer is 4 bytes, index 5 starts exactly `Base + 20 bytes` away.

How dynamic arrays resize themselves under the hood.

When pushing an element to a full std::vector, the OS must allocate a brand new, larger contiguous block of memory (usually 2x size), copy all old elements over, and delete the old block.

Because resizing doubles the capacity, the heavy O(N) copy penalty happens exponentially less often. Across N insertions, the average cost per insertion smooths out to exactly O(1) amortized time.

🛠️ Block 2 — Basics & Pointers

⏱️ 3 Hours

Given an array, return indices of the two numbers that add up to a specific target.

O(N) Time | O(N) Space via HashMap

vector<int> twoSum(vector<int>& nums, int target) {
    unordered_map<int, int> m;
    for(int i=0; i<nums.size(); i++) {
        int complement = target - nums[i];
        if(m.count(complement)) {
            return {m[complement], i};
        }
        m[nums[i]] = i; // Save value and index
    }
    return {};
}
Theory: By trading space for time, we cache our past iterations. Rather than an N2 nested loop, we ask the hashmap "Have I seen what I need?" in O(1) time.

Remove all instances of `val` in-place and return the new length.

O(N) Time | O(1) Space via Fast/Slow Pointers

int removeElement(vector<int>& nums, int val) {
    int insertPos = 0; // Slow pointer
    for (int i = 0; i < nums.size(); i++) { // Fast pointer
        if (nums[i] != val) {
            nums[insertPos] = nums[i];
            insertPos++;
        }
    }
    return insertPos;
}

Move all 0s to the end of it while maintaining the relative order of the non-zero elements.

O(N) Time | O(1) Space

void moveZeroes(vector<int>& nums) {
    int left = 0;
    for (int right = 0; right < nums.size(); right++) {
        if (nums[right] != 0) {
            swap(nums[left], nums[right]);
            left++;
        }
    }
}

⚙️ Block 3 — Math & Algorithms

⏱️ 3 Hours

Find the element that appears more than ⌊n / 2⌋ times. Solve in strictly O(1) space.

Boyer-Moore Voting Algorithm

int majorityElement(vector<int>& nums) {
    int count = 0, candidate = 0;
    for (int num : nums) {
        if (count == 0) candidate = num;
        if (num == candidate) count++;
        else count--;
    }
    return candidate;
}

Find the index where sum of elements strictly to the left equals sum of elements strictly to the right.

O(N) Prefix Math

int pivotIndex(vector<int>& nums) {
    int totalSum = 0, leftSum = 0;
    for (int x : nums) totalSum += x;
    
    for (int i = 0; i < nums.size(); i++) {
        if (leftSum == totalSum - leftSum - nums[i]) {
            return i;
        }
        leftSum += nums[i];
    }
    return -1;
}

Given integer array, return array such that answer[i] is equal to product of all elements except nums[i]. Without division division operator.

Two-Pass Running Prefix & Suffix Products

vector<int> productExceptSelf(vector<int>& nums) {
    int n = nums.size();
    vector<int> ans(n, 1);
    
    // Left pass
    int leftProd = 1;
    for(int i = 0; i < n; i++) {
        ans[i] = leftProd;
        leftProd *= nums[i];
    }
    
    // Right pass
    int rightProd = 1;
    for(int i = n - 1; i >= 0; i--) {
        ans[i] *= rightProd;
        rightProd *= nums[i];
    }
    return ans;
}

🔥 Block 4 — Shifting & Sorting

⏱️ 3 Hours

Rotate list to the right by k steps in O(1) extra space.

The Triple-Reverse Trick

void rotate(vector<int>& nums, int k) {
    k = k % nums.size(); // Standardize K
    // 1. Reverse entire array
    reverse(nums.begin(), nums.end());
    // 2. Reverse first K
    reverse(nums.begin(), nums.begin() + k);
    // 3. Reverse remaining
    reverse(nums.begin() + k, nums.end());
}

Sort array of 0s, 1s, and 2s strictly in one pass (Dutch National Flag problem).

Dutch National Flag (Tri-Partition)

void sortColors(vector<int>& nums) {
    int low = 0, mid = 0, high = nums.size() - 1;
    while(mid <= high) {
        if(nums[mid] == 0) {
            swap(nums[low], nums[mid]);
            low++; mid++;
        }
        else if(nums[mid] == 1) {
            mid++;
        }
        else { // It's 2
            swap(nums[mid], nums[high]);
            high--; // Do NOT increment mid here! Value swapped might be 0
        }
    }
}
