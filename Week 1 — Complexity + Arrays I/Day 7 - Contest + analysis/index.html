Lucky Prefix
Prefix Sum · long long
★ 900 500 pts Not Started

Given an array a of n positive integers. Find the minimum index k (1-indexed) such that the sum of the first k elements is strictly greater than the sum of the remaining n−k elements. If no such k exists, print -1.

Input
5
1 2 3 4 5
Output
3
Constraints
n ≤ 2×10⁵
a[i] ≤ 10⁹

💡 Key Insight:

Build prefix sum. For index k: prefix[k] > total - prefix[k] → 2*prefix[k] > total. Scan left to right.

🚨 Critical: Use long long! Sum reaches 2×10¹⁴.

long long total=0; for(auto x:a) total+=x;
long long prefix=0;
for(int k=1;k<=n;k++){
    prefix+=a[k-1];
    if(prefix*2 > total){ cout<<k; return 0; }
}
cout<<-1;

Maximum Flip Gain
Kadane Min Subarray
★ 1400 1250 pts Not Started

Choose exactly one contiguous subarray and negate all its elements. Maximize total array sum.

Input
5
-5 2 3 -1 4
Output
13
Constraints
n ≤ 2×10⁵
|a[i]| ≤ 10⁹
Negate [-5]: total = -5+2+3-1+4 = 3. New = 5+2+3-1+4 = 13.

💡 Key Insight: total − 2 × min_subarray

Negating [l,r]: new_total = total − 2×sum(l,r). Minimize sum(l,r) = run Kadane for min.
Works for all cases: if min ≥ 0 (all positive), we still negate the min single element.

// Kadane for minimum:
long long curr=a[0], mn=a[0];
for(int i=1;i<n;i++){ curr=min(a[i],curr+a[i]); mn=min(mn,curr); }
cout << total - 2*mn;

Input

-1 3 -2 4
Output

Constraints

[3] + [4] = 7. Or [-1,3,-2,4] requires split: [3]+[4]=7.

💡 Precompute L[i] and R[i], then max(L[i]+R[i+1])

L[i] = best subarray sum in a[0..i] (running max Kadane forward).
R[i] = best subarray sum in a[i..n−1] (running max Kadane backward).

// Backward similarly for R[]
