# Day 7: Week 1 Contest — "Array Siege"

## Overview
This is the **capstone contest** for Week 1. All 7 problems test the topics covered Days 1–6. It simulates a real Codeforces Div. 2 round in structure and difficulty.

## Contest Details
| Field | Value |
|---|---|
| Duration | 2.5 hours |
| Problems | A – G (7 total) |
| Style | Codeforces Div. 2 |
| Topics | Complexity · Arrays · Prefix Sum · Diff Array · Kadane · Edge Cases |

## Problem Overview
| # | Title | Difficulty | Concept |
|---|---|---|---|
| A | Lucky Prefix | ★ 900 | Prefix sum + long long |
| B | Storm Coverage | ★ 1100 | Difference array |
| C | Zero Hunters | ★ 1200 | Prefix sum + HashMap (freq[0]=1) |
| D | Maximum Flip Gain | ★ 1400 | Kadane min subarray |
| E | Dominance Count | ★ 1600 | Prefix + inversion count |
| F | Circular Champion | ★ 1700 | Circular Kadane (complement trick) |
| G | Double Harvest | ★ 1900 | Two-pass Kadane precomputation |

## How to Use
1. Open `index.html` — start the 2.5h timer.
2. Solve problems **in order** — don't skip.
3. After each problem, record your approach in the submission box.
4. Reveal editorial only after attempting or after contest ends.
5. Complete post-contest analysis in `todo.md`.
