# Day 2: Palindromes & Substrings

## Overview
Welcome to Day 2! Today we tackle palindromes—strings that read the same forwards and backwards—and dive deep into algorithms dealing with contiguous segments of strings (substrings). These are incredibly common in technical interviews.

## Learning Goals
By the end of today, you will be able to:
- Instantly verify palindromes using two pointers.
- Apply the **Expand Around Center** technique for identifying palindromic substrings.
- Use **Sliding Window** to efficiently evaluate substrings without redundant work.
- Grasp the foundations of **Palindrome DP**, recognizing overlapping subproblems.
- (Advanced) Understand the basics of **Manacher's Algorithm** for finding the longest palindromic substring in $O(N)$ time.

## Prerequisites
- Array iteration and basic Two Pointers (from Week 2).
- Fundamentals of ASCII mapping and hash maps (from Week 3).
- String basic operations (from Week 4 - Day 1).
