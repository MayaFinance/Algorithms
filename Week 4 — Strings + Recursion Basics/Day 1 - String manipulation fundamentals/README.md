# Day 1: String Manipulation Fundamentals

## Overview
Welcome to the start of Week 4! Strings are ubiquitous in programming interviews. This module rebuilds your foundations from the ground up: mapping strings to memory, manipulating characters safely, avoiding accidental complexity, and mastering frequency and two-pointer basics.

## Learning Goals
By the end of today, you will be able to:
- Predict string mutability and space constraints perfectly.
- Avoid $O(N^2)$ traps during string building.
- Build and trace frequency counter arrays ($O(1)$ space).
- Apply Two-Pointer logic to string edges and subsets.
- Understand prefix techniques and fundamental pattern matching boundaries.

## Prerequisites
- Firm grasp of array manipulations (Week 2).
- Basic understanding of ASCII indexing (e.g., `'c' - 'a'`).
